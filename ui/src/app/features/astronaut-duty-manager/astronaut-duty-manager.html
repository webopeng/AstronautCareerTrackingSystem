Business Rules Reference

Assignment Rules

  • • One current duty per person at a time
  • • Previous duty ends day before new assignment
  • • Current duties have no end date

Retirement Rules

  • • 'RETIRED' title marks career end
  • • Career end date = retirement start - 1 day
  • • Retired personnel have no current duty
